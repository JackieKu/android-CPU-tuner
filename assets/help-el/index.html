Βοήθεια του CPU tuner

The English help pages have been updated.
Consider using the Enlish version Το CPU tuner χρησιμοποιεί μια έννοια των Εναυσμάτων και προφίλ. Τα εναύσματα ενεργοποιούνται στο τηλέφωνό σας όταν αυτό εισέρχεται σε μια προκαθορισμένη κατάσταση. Εφαρμόζεται, στη συνέχεια, ένα προκαθορισμένο προφίλ. Οι καταστάσεις ορίζονται από το επίπεδο της μπαταρίας και την κατάσταση του τηλεφώνου σας (κλειδωμένο, ενεργό ή στο ρεύμα). Προφίλ είναι ουσιαστικά μια σειρά από πολιτικές που εφαρμόζονται στο τηλέφωνο. Πράττοντας αυτό, το CPU tuner δε τρέχει συνεχώς δημιουργώντας έτσι μόνο ελάχιστο φόρτο συστήματος..
Αυτή η βοήθεια περιέχει επίσης ένα FAQ που μπορεί να βρείτε ενδιαφέρον. Μπορείτε να βρείτε περισσότερες πληροφορίες για το θέμα της εφαρμογής σε αυτή τη συλλογή συνδέσμων.

ΣΗΜΕΙΩΣΗ: Για να λειτουργήσει σωστά, το CPU tuner χρειάζεται ένα τηλέφωνο με δικαιώματα υπέρ-χρήστη και αυτά τα δικαιώματα να δοθούν στην εφαρμογή από σας. Πρέπει να γνωρίζετε ότι ορισμένες ROMs δεν υποστηρίζουν προσαρμογές τέτοιων βαθέων παραμέτρων του συστήματος!
