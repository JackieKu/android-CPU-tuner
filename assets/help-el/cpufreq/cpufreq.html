CPU tuner / cpufreq

Το CPU Tuner σας βοηθά στη δημιουργία μιας αξιοπρεπούς διαμόρφωσης δυναμικής αλλαγής στην λειτουργία της CPU για το κινητό σας τηλέφωνο.. Σας επιτρέπει να θέσετε CPU governor και όρια συχνότητας μέσω εύκολου γραφικού περιββάλοντος. Στο παρασκήνιο, χρησιμοποιεί τον πυρίνα του Linux: περιβάλλον cpufreq.
