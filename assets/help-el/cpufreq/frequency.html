Συχνότητα CPU

Η ταχύτητα ρολογιού του επεξεργαστή σας ("πόσο γρήγορος είναι")
Όσο μεγαλύτερη η τιμή, τόσο πιο γρήγορα θα ανταποκρίνεται το τηλέφωνο σας. Λάβετε υπόψη ότι οι υψηλές συχνότητες καταναλώνουν τη μπαταρία σας πιο γρήγορα, καθώς η CPU δουλεύει περισσότερο ("σκληρότερα") και χρειάζεται υψηλότερη τάση πυρήνα για να λειτουργήσει σωστά. Επίσης, ο υπερχρονισμός οδηγεί σε περισσότερη παραγωγή θερμότητας, που θα μπορούσε να προκαλέσει βλάβη στο τηλέφωνο. Έχετε προειδοποιηθεί! Το αντίστροφο νόημα είναι ότι το τηλέφωνο θα είναι πιο αργό όταν του κάνετε υποχρονισμό, αλλά θα χρησιμοποιεί λιγότερη ενέργεια κάτι που οδηγεί σε παρατεταμένη διάρκεια ζωής της μπαταρίας. Το να υποχρονίσετε σε τιμές κάτω από 400MHz μπορεί να προκαλέσει εγκλωβισμό του τηλεφώνου σας και να χρειαστεί να του βγάλετε τη μπαταρία για να ξεκολλήσει. Αυτό είναι ένας απ τους λόγους που συχνότητες κάτω απ την προαναφερόμενη τιμή είναι διαθέσιμες μόνο για "έμπειρους χρήστες". Αν είστε εδώ για να παρατείνετε το χρόνο λειτουργίας της μπαταρίας σας, ίσως να θέλετε να ρίξετε μια ματιά σε άλλους γνωστούς καταναλωτές μπαταρίας.
