CPU governors / Κυβερνήτες της CPU

Οι CPU governors είναι λογισμικό μέσα στο πυρίνα που ελέγχει τη ταχύτητα της CPU μέσα από διάφορα κριτήρια. Σύμφωνα με το πυρίνα του Linux οι ακόλουθοι κυβερνήτες είναι διαθέσιμα:
  • ondemand
    Αλλάζει δυναμικά τη συχνότητα με βάση τη χρήση του επεξεργαστή, με έμφαση στις επιδόσεις.
    Περισσότερες πληροφορίες...
  • conservative
    Αλλάζει δυναμικά τη συχνότητα με βάση τη χρήση του επεξεργαστή, με έμφαση στην οικονομία μπαταρίας.
    Περισσότερες πληροφορίες...
  • performance
    Κλειδώνει τη συχνότητα στη μέγιστη δυνατή.
    Περισσότερες πληροφορίες...
  • powersave
    Κλειδώνει τη συχνότητα στην ελάχιστη δυνατή.
    Αυτός ο κυβερνήτης ενδέχεται να παρουσιάσει ανεπιθύμητη συμπεριφορά, καθώς ένα μεγάλο φόρτο εργασίας έχει μεγαλύτερο χρόνο εκτέλεσης από ό,τι θα έπρεπε, χρησιμοποιώντας τον κυβερνήτη OnDemand
    Περισσότερες πληροφορίες...
  • userspace
    Τοποθετεί τη συχνότητα αυτόματα σ αυτή που δώσατε.
    ΔΕΝ ΠΡΕΠΕΙ ΝΑ ΧΡΗΣΙΜΟΠΟΙΕΙΤΑΙ αν δεν ξέρετε τι κάνετε!
    Περισσότερες πληροφορίες...
  • interactive
    Σχεδιασμένο για χαμηλό χρόνο απόκρισης σε περίπτωση ξαφνικής απαίτησης μεγάλης επεξεργασίας. Η κλιμάκωση γίνεται όταν η συσκευεί βγαίνει από αδράνεια αντί μέσω polling.
