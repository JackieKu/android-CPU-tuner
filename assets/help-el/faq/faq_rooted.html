Χωρίς δικαιώματα υπερ-χρήστη;

Αν έχετε αποκτήσει δικαιώματα υπερ-χρήστη στο τηλέφωνο, μπορεί να μη προσέξατε να δώσετε αυτά τα δικαιώματα στο CPU tuner όταν το ανοίξατε.
Κάντε έλεγχο στην εφαρμογή Υπερ-χρήστη και δείτε αν το CPU tuner έχει έγκριση.
Εαν δεν αποκτήσατε δικαιώματα υπερ-χρήστη θα χρειαστεί να κάνετε τη διαδικασία, αν θέλετε να χρησιμοποιήσετε το CPU tuner.
Θα σας συμβούλευα να μη το κάνετε ωστόσο!
Η διαδικασία απόκτησης δικαιωμάτων υπερ-χρήστη, ενδεχομένως, να σας δημιουργήσει πρόβλημα στο τηλέφωνο/λογισμικό. Οπότε θα πρέπει να έχετε σοβαρό λόγο για να το κάνετε.
Προσωπικά, δε θεωρώ το CPU tuner έναν "σοβαρό λόγο"!
Σκεφτείτε επίσης, ότι αν αποκτήσετε δικαιώματα υπερ-χρήστη, ακυρώνετε την εγγύηση του τηλεφώνου σας!
