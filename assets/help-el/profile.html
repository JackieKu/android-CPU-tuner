Consider using the Enlish version Τα προφίλ είναι ρυθμήσεις σε επίπεδο συστήματος εφαρμόσιμα από εναύσματα. Για την ώρα (v. 1.0) υπάρχουν δύο κατηγορίες, που μπορείτε να ρυθμήσετε:
  • Ρυθμήσεις στη CPU
    Αυτές οι ρυθμήσεις ορίζουν τη ταχύτητα της CPU("πόσο γρήγορη είναι") και την ενεργειακή κατανάλωση.
  • Υπηρεσίες
    Αυτές οι ρυθμήσεις μπορούν να ενεργοποιήσουν ή/και να απενεργοποιήσου κάποιες υπηρεσίες.
    Όπως ειπώθηκε και στο καταναλωτές ενέργειας είναι συνετό να απενεργοποιείτε το συγχρονισμό, το ασύρματο κλπ σε κάποιες καταστάσεις.
    Μπορείτε να αφήσετε [την αναφερόμενη υπηρεσία] χωρίς αλλαγές, αλλά μπορείτε επίσης να ενεργοποιήσετε η απενεργοποιήσετε υπηρεσίες με εναύσματα, επίσης μπορείτε να επιστρέψετε πίσω [στην] προηγούμενη κατάσταση που ενεργοποιεί την κατάσταση της υπηρεσίας που είχε πριν από την τελευταία αλλαγή.
    Αν παρακάμψετε χειροκίνητα αυτές τις ρυθμίσεις για παράδειγμα αν το CPU tuner κλείσει το συγχρόνισμο στο 20% της μπαταρίας κι εσείς θέλετε να συγχρόνισετε το ημερολόγιο, αυτή η ρύθμιση θα μείνει ανέγγιχτη από το CPU tuner μέχρι να ενεργοποιηθεί κάποιο άλλο έναυσμα το οποίο θα τροποποιεί τη κατάσταση της υπηρεσίας αυτής.
