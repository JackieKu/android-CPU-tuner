Settings: System

  • Check Capabilities
    Check if your devices (that is you kernel) support cpu tuner.
  • Battery hot temperature
    Set a custom temperature (in Â°C) when the batter hot profile gets triggered.
    If not set battery hot gets triggered by a system event.
  • Show Userspace Governor
    Not recommended since to userspace governor does not do any battery saving.
  • Min displayed frequency
    The minimum displayed cpu frequency for the maximum cpu frequency. Default is 400 MHz.
  • Allowed CPU frequencies
    A list of frequencies (in Hz not MHz) separated by spaces, which are used a cpu frequencies if the system does not report any.
  • Install as System App
    Enables GPS switching, but is not supported needs the beta key and power user mode.
    I (the developer) do hardly ever use it.
