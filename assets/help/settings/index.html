Settings: Main

  • Enable cpu tuner
    Enable or disable cpu tuner (i.e. start or stop). When cpu tuner is stopped it keeps the last setting applied.
  • Configurations
    Backup and restore as well as autoload configurations (i.e. a set of the profiles, triggers and virtual governors).
  • User interface
    Settings which influence the interface of cpu tuner.
  • Backend
    Settings which influence the way cpu tuner works.
  • Extensions
    Extensions to cpu tuner that cost a little money.
  • Buy me a beer
    Does nothing, except it gives you a possibility to spend money and make me a little bit happy.
    But you even make me happier if you participate on https://sourceforge.net/projects/cputuner/ and answer questions of new users or participate in any way that fits you best.
  • Various
    Varous informations like licenses, changelog or version of cputuner.
