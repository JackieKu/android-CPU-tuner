Settings: Service switches

  • Manual service switches
    If checked this allows a service being changed manually (i.e. outside cpu tuner) without cpu tuner switching it back on the next profile change.
    Or more exactly cpu tuner will only switch a service if it is in the same state as it was switched to by cpu tuner. Pulsed services and the back to previous service switch are not affected.
  • Netmode on wifi connected
    Switch to a different network mod (e.g. 2G or 3G) if the device is connected to a wifi network.
  • Switch wifi if connected
    If check cpu tuner will switch wifi (off) even if the device is connected to a wireless network.
Pulsed services will be switched off for some time, then automatically switched on for a period. After this the cycle begins again.
All pulsed services will be switch on and off at the same time.
  • Service on time
    The time in minutes a service will be on when pulsed.
  • Service off time
    The time in minutes a service will be off when pulsed.
  • Pulse mobiledata on wifi
    If not checked the mobiledata connection will not be pulsed while connected to a wireless network.
