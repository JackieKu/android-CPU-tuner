Settings: User interface

  • Choose language
    Set cpu tuner to a different language than your device.
  • Experiance level
    This changes the possible settings of cpu tuner, things are a little easier or more complicated.
  • Calculate power usage
    The power usage can be calculated based on current or average values.
  • Profile switch log size
    Cpu tuner keeps track of the last profile switches. If you enter a bigger log size cpu tuner will use more memory Ssince the log is kept in memory.
  • Add to status bar
    Check to add the cpu tuner icon to the status bar and a entry in the notification area displaying the current profile and launching cpu tuner if clicked.
  • Statusbar notifications
    If checked each profile change will be show in the statusbar.
