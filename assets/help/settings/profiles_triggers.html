Settings: Profiles and triggers

  • Enable profiles
    Check this to let cpu tuner automatically manage you profiles.
    Unchecking it mean to disable cpu tuner.
  • Call in Progress Profiles
    Adds a profile which is applied while the phone is ringing or during a phone call.
  • Enable virtual governors.
    Check to enable the virtual governors,
  • Reset Profiles and Triggers
    Resets all profiles, triggers and virtual governors to their default values.
