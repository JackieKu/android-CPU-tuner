capability check

You can have CPU tuner check your device's capabilities in regard to tuning it's CPU. Please note that CPU tuner does not support phones, but it needs a kernel with support for the cpufreq interface. CPU tuner produces good results if at least one of the following governors is working.
  • ondemand governor
    That is a ondemand governor that allows reading and writing to the up threshold
  • conservative governor
    That is a conservative governor that allows reading and writing to the up and down threshold.
For more information you can go to the CPU tuner capability check forum. If you looking for more information on rooting or getting a new ROM or kernel checkout the xda-developers forums or CyanogenMod.
