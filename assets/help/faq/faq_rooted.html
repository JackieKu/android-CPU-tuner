There are two cases that are possible:

The device is rooted

When opening CPU tuner you get asked if you want to allow root access and you have to grant root access in oder for CPU tuner to work correctly.
There should be an app called Superuser open it and check if CPU tuner is granted root access.

The device is NOT rooted...

Howto root the device?

You did not root your device.
In this case I would NOT advice you to root the Device!
Rooting a phone has potential to break it and unless you have very good reasons (I would not consider CPU tuner as a good reason.) and are confident about it, I would not do it.
You will void you warranty and potentially nobody is going to help you if something is going wrong.
