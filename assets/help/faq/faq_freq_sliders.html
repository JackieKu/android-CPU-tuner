The sliders do not move or are locked to 0.

The sliders itself work but your kernel does not accept any frequency changes.

There are to possibilities:

  • The sliders are locked to 0
    The kernel of you device does not allow the frequency to be read (and not written as well).
  • The sliders are locked to a values
    Cpu tuner is able to read the current settings but the kernel does not allow it to be changed.
