cpu tuner help

CPU tuner uses a concept of triggers and profiles.
Depending on the state of the device a trigger applies a profile.

The state of the device consists of the battery level and whether the device is locked or on power.
The profile is a set of policies that will be applied if a profile gets activated (by a trigger).

CPU tuner does not constantly run and poll, but it gets activated by events sent by the android system.
