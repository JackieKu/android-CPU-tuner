I selettori non si muovono e/o sono bloccati a 0

I selettori di per sé funzionano, ma il kernel del tuo telefono non supporta o accetta cambi di frequenza. Cause possibili:
  • selettori bloccati a 0
    Il kernel del tuo telefono non permette di leggere (o scrivere) le frequenze.
  • selettori bloccati a un valore ≠ 0
    Il kernel del tuo telefono permette di leggere ma non di scrivere le frequenze.
Puoi verificare le funzionalità del tuo telefono.
