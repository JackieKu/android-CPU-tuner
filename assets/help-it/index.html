cpu tuner help

CPU tuner funziona tramite i trigger e i profili. I trigger vengono attivati sul tuo telefono inserendo uno stato definito in precedenza. Quindi applicano un profilo definito in precedenza. Gli stati sono definiti a seconda del livello della batteria e lo stato del telefono (schermo bloccato, attivo o alimentato dalla rete elettrica). Profiles are essentially a set of policies that are applied to the phone. While doing so, CPU tuner runs event-driven. I.e. it does not run constantly, thus creating only minimal system load on it's own.
This help also includes a FAQ which you might find helpful/interesting. You can find further info on the application's topic in this link collection.
